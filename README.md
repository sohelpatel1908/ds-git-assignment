## **Commands UsedCommand**
1. **`git init`**: Initialize a new local repository
2. **`git status`**: Check the state of the working directory
3. **`git add <file>`**: Stage changes for a commit
4. **`git commit -m ""`**: Save staged changes with a descriptive message
5. **`git checkout -b <name>`**: Create and switch to a new branch
6. **`git merge <name>`**: Combine changes from one branch into another
7. **`git push`**: Send local commits to the remote GitHub server
